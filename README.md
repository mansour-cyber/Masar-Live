# Masar Live

نسخة التشغيل العامة من مشروع **مسار**، مخصصة للمعاينة والاختبار.

- `/` صفحة هبوط بيعية موجهة للشركات الصغيرة والمتوسطة.
- `/app/` محرر العمليات والحوكمة الفعلي.
- `editor.html` يحافظ على رابط الدخول السابق وينقل المستخدم إلى المحرر.

المصدر الأساسي وخطة التطوير والتوثيق الهندسي محفوظة في مستودع خاص منفصل.
